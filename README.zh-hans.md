# Thebes
> ***Thebans（底比斯人）*** 为 ***Sparta（斯巴达）*** 不可战胜的神话画上了句号。

`Thebes` 是一个简单的 HTTP 回显服务器，用于接收和记录来自修改过的战地 1 Sparta / Twinkle 客户端的信息，以供研究和本地化之用。

更多信息请参考 ParaTranz 上的 **[原始讨论](https://paratranz.cn/projects/8862/issues/16874)**。
